# digital_recognition
# 手写数字识别程序

# Run
在linux下使用命令 'python main.py' 运行

# 需要安装一些必要的库
##tkinter
>$sudo apt-get install python-tk
##opencv
>$sudo pip install opencv-python
##numpy
>$sudo pip install numpy
##scipy
>$sudo pip install scipy
##PIL
>$sudo pip install pillow
